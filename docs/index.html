API Docs

api/cities

GET

Returns a list of all the cities in the database belonging to the user's top city list.

Request Headers

Authorization: a token used to identify the user, gotten from the auth API.

Response Body

{ "area": "New York city", "rank": 1, "userHasVisited": false, "id": 0 }

POST

Accepts an array of new cities which are added to the database and ranked automatically

Request Headers

Authorization: a token used to identify the user, gotten from the auth API.

Request Body

{ "area": "example area name", "userHasVisited": false }
